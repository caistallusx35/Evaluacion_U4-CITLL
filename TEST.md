# Investigación y Respuestas

### 1. ¿Qué es un archivo Markdown y para qué sirve?
Es un lenguaje ligero que permite aplicar el formato a un texto (como negritas, títulos o tablas) usando caracteres basicos. Sirve principalmente para crear o escribir en blogs o gestionar proyectos en GitHub de forma que el texto sea legible tanto en crudo como renderizado.

### 2. ¿Cuál es el propósito del archivo README.md en un repositorio?
*El propósito del archivo README.md es proporcionar una descripción detallada y clara de lo que contiene el repositorio, explicando su funcionamiento, instalación y cualquier otra información relevante para que otros usuarios o colaboradores entiendan el proyecto rápidamente.*

### 3. Sintaxis de Markdown

| Elemento | Cómo se escribe (Sintaxis) |
| :--- | :--- |
| **Títulos** | `# H1`, `## H2`, `### H3` |
| **Listas Ordenadas** | `1. Primer elemento` |
| **Listas Desordenadas** | `- Elemento simple` |
| **Letra Cursiva** | `*Texto en cursiva*` |
| **Letra Negrita** | `**Texto en negrita**` |
| **Insertar un enlace** | `[Texto del link](URL)` |
